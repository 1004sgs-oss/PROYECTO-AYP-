Descripción Base de datos 2

Hospitalización 2020



Descripción general



¿Qué es la base de datos?



Esta base de datos contiene información relacionada con registros de hospitalización en Colombia durante el año 2020. Incluye datos asociados a la atención hospitalaria y variables utilizadas para análisis estadísticos y administrativos del sector salud.



Contexto y propósito



La información permite analizar el comportamiento de los servicios hospitalarios, apoyar investigaciones en salud pública y realizar estudios relacionados con atención médica y hospitalización.



¿Cómo se obtuvieron los datos?



Los datos fueron recopilados mediante registros administrativos del sistema de salud y publicados en el portal de Datos Abiertos Colombia.



Fuente / https://www.datos.gov.co/dataset/2020\_Hospitalizacion/ifk7-8jyr/about\_data
